---
title: "ReadyAction 앱"
---

ReadyAction 은 Daven AI 기반위에 만들어진 독립형 앱입니다. 전문 영상물을 쉽게 만들기 위한 앱입니다.

1. Action Agent : [ReadyAction.ai](http://ReadyAction.ai) 의 메인 페이지에 있는 Action Agent 에 만들고 싶은 컨텐츠 내용을 입력하면 해당 생성 모드와 설정값을 제안해줍니다.
2. 다양한 생성 모드 선택 : 상단에 있는 다양한 분류에서 사용자가 생성 모드를 선택해서 시작할수도 있습니다. 예를 들면 Film, Music Video, Promo, 다양한 형식의 숏폼 비디오 등등이 있습니다.
3. 생성 방식 선택 : ReadyAction 은 Agent 를 통한 자동화 생성, 사용자가 개입해서 만드는 반자동 생성, 사용자가 처음부터 수동으로 만드는 매뉴얼 모드가 있습니다. 각 방식의 선택은 각 기능을 선택하면 진행하면서 사용자가 결정할 수 있습니다. 
4. 캐릭터 : ReadyAction 에서 만든 캐릭터는 Daven 과 상호 호환됩니다. 
5. ReadyAction App은 토큰은 Daven 과 상호 호환됩니다.
