## 토큰 쿠폰

토큰 쿠폰은 멤버쉽은 유지하면서 토큰이 충전되는 쿠폰입니다.

설정 \> 구독 \> 토큰 쿠폰이 있으신가요? \
에서 쿠폰을 넣으시면 됩니다. ( 유효하지 않은 쿠폰일경우 적용이 되지 않으며 적용 불가 사유가 표시 됩니다. )

## 멤버십 쿠폰

멤버쉽 쿠폰은 멤버쉽을 제공해주는 쿠폰입니다.

[www.daven.ai/pricing](http://www.daven.ai/pricing) 에서 해당 멤버쉽을 선택하시고 \
예) Plus 4000 Token\
다음 페이지에서 Promo Code 를 누르시고 입력하셔야 합니다.

쿠폰의 상품과 선택한 멤버쉽이 일치해야 쿠폰이 적용 됩니다. ( 유효하지 않은 쿠폰일경우 적용이 되지 않으며 적용 불가 사유가 표시 됩니다.)
